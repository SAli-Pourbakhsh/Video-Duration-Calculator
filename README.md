\# Video Duration Calculator \& Time Manager



A professional, CLI-based tool designed to calculate the total duration of video files. It supports both \*\*manual time entry\*\* (smart parsing) and \*\*recursive directory scanning\*\*. Ideal for students, researchers, and video editors who need to manage course durations or project timelines.



!\[Python](https://img.shields.io/badge/Python-3.8%2B-blue)

!\[License](https://img.shields.io/badge/License-MIT-green)

!\[Status](https://img.shields.io/badge/Status-Stable-brightgreen)



\## ‚ú® Key Features



\* \*\*Smart Parsing:\*\* Automatically detects formats like `MM:SS` or `HH:MM:SS`.

\* \*\*Directory Scan:\*\* Recursively scans folders to find all video files and calculate total duration.

\* \*\*Undo Functionality:\*\* Easily remove the last entered entry in manual mode.

\* \*\*Playback Speed Analysis:\*\* Shows how long it takes to watch videos at 1.25x, 1.5x, 2.0x speeds.

\* \*\*Format Flexibility:\*\* Handles various input styles (e.g., copy-pasting from messy text sources).

Choose the method that suits you best:



\### Method 1: No Installation Required (Easiest)

If you don't have Python installed or just want a quick start:

1\.  Go to the \[\*\*Releases\*\*](https://github.com/SAli-Pourbakhsh/Video-Duration-Calculator/releases) page.

2\.  Download the latest `VideoCalculator.exe`.

3\.  Double-click to run.



\### Method 2: The Automatic Launcher (Recommended for Windows)

If you have Python installed but don't want to manage dependencies manually:

1\.  Download the source code (Clone or Download ZIP).

2\.  Double-click on `run\_calculator.bat`.

3\.  The script will automatically install necessary libraries (`moviepy`, `colorama`) and launch the tool.



\### Method 3: For Developers (Source Code)

If you prefer running via IDE (VS Code, PyCharm) or terminal:



1\.  Clone the repository:

&nbsp;   ```bash
