## üìñ Overview
**Video Duration Calculator** is a robust CLI-based tool designed to calculate the total duration of video files. Whether you have a messy list of timestamps copied from a website or a folder full of video tutorials, this tool sums them up instantly.

It supports **Smart Parsing**, **Recursive Directory Scanning**, and provides **Playback Speed Analysis**.

---

## ‚ú® Key Features

### ‚úÖ **Smart Parsing:** Automatically detects formats like `MM:SS` or `HH:MM:SS`.
### ‚úÖ **Directory Scan:** Recursively scans folders to find all video files and calculate total duration.
### ‚úÖ **Undo Functionality:** Made a mistake? Easily undo the last entry in manual mode.
### ‚úÖ **Playback Speed Analysis:** Shows how long it takes to watch videos at **1.25x**, **1.5x**, **2.0x** speeds.
### ‚úÖ **Robust Input:** Handles copy-pasting from messy text sources seamlessly.
